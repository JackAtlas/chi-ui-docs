## API

### Icon 属性

| 名称   | 类型       | 说明           | 默认值 |
| ------ | ---------- | -------------- | ------ |
| class  | `string`   | css 类名       |        |
| height | `number`   | 画布高度（px） | `300`  |
| images | `string[]` | 图标 url 数组  |        |
| width  | `number`   | 画布宽度（px） | `300`  |
